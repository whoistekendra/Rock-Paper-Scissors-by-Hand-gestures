✊✋✌️ Rock Paper Scissors by Hand Gestures

A Python-based game that uses real-time hand gesture recognition via webcam to play Rock Paper Scissors against the computer using computer vision.

🎮 Features


Real-time hand detection using your webcam
Gesture recognition for Rock ✊, Paper ✋, and Scissors ✌️
Instant computer move generation
Score tracking across rounds
Simple and interactive UI overlay


🛠️ Tech Stack


Python 3
OpenCV – for video capture and image processing
MediaPipe – for hand landmark detection


📦 Installation


Clone the repository


bash   git clone https://github.com/whoistekendra/Rock-Paper-Scissors-by-Hand-gestures.git
   cd Rock-Paper-Scissors-by-Hand-gestures


Install the required dependencies


bash   pip install opencv-python mediapipe


Run the game


bash   python file.py
